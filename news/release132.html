Release Announcement Bitcoin Classic 1.3.2

August 1st, 2017.

Bitcoin Classic updates the 1.3 version to add a migration strategy.

This client is fully compatible with the Bitcoin Cash spin-off from Bitcoin, but using the default settings we ignore it. You can change the config-file, or we point you to the specialized UAHF version of Classic.

Details;

  • 233605e0 Add new feature; blocksdatadir config
  • a5bca35e Make a data-migration path. (copying databases)
  • 056532df Disable headers code for now.
  • d7ca53ca Remember the xthin and cash nodes between runs.
  • bd29c605 Restore sanity; asserts are for debug builds only

And various minor bugfixes and updates.

For more help with installation, and instructions how to download please visit bitcoinclassic.com/gettingstarted.html
