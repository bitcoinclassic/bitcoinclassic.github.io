Download Bitcoin Classic

Since the first of August there are two main versions of Bitcoin. All your old coins are available on both.
The first is the legacy chain, Bitcoin. Using Bitcoin Classic wallet you can fully transact BTCs.
The second is the Bitcoin Cash chain, which started on August 1st. This chain removes a centralized decision making team from power and immediately followed through with a long expected upgrade to allow bigger blocks.

Please see the pre-configured downloads below as a way to select which chain you want to follow.

 

Bitcoin Classic 1.3.8 for Bitcoin Cash

Bitcoin Classic updates the 1.3 version with various bugfixes.

This release by default follows the Bitcoin Cash chain created in August, this chain supports up to 8MB blocks and support is growing every day.

For details on what is included in this release, please see the release notes.
